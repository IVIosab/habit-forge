## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed on your machine:

- **Node.js** (version 22 or higher)
- **npm** (version 10 or higher)

### Installation

To install the project, follow these steps:

1. Install the dependencies:
   ```bash
   npm install
   ```
2. Run it:
   ```bash
   npm run dev
   ```
